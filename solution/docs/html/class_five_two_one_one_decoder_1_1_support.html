static byte[] FiveTwoOneOneDecoder.Support.GetBytes ( byte[]  data,
int  startIx,
int  length

static

Copies a subset of bytes from the specified byte array, data, into a new byte array. The first byte copied is specefied by startIx and the number of bytes to copy is specified by length.

Example:

byte[] data = { 0x01, 0x02, 0x03, 0x04, 0x05, 0x06 };
byte[] subdata = Support.GetBytes(data, 2, 2);
Console.WriteLine(Support.ToHex(subdata));
Output:
0x0304
Parameters
dataThe bytes to copy.
startIxThe first byte to copy.
lengthThe number of bytes to copy.
Returns
A subset of the bytes from the specified byte array.
